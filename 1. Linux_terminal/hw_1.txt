1) Посмотреть где я
pwd
2) Создать папку
mkdir dir_hw1
3) Зайти в папку
cd dir_hw1/
4) Создать 3 папки
mkdir folder_1 folder_2 folder_3
5) Зайти в любоую папку
cd folder_2/
6) Создать 5 файлов (3 txt, 2 json)
touch file_1.txt file_2.txt file_3.txt file_4.json file_5.json
7) Создать 3 папки
mkdir d_4 d_5 d_6
8. Вывести список содержимого папки
ls -la
9) + Открыть любой txt файл
vim file2.txt
10) + написать туда что-нибудь, любой текст.
Клавиша I(insert) режим редактирования, пишем текс

11) + сохранить и выйти.
нажимаем Esc, ставим двоеточие wq 

12) Выйти из папки на уровень выше
cd ..
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку.
mv folder_2/file_1.txt folder_2/file_2.txt folder_1/
или по отдельности 
				mv folder_2/file_1.txt folder_1/
				mv folder_2/file_2.txt folder_1/
14) скопировать любые 2 файла, которые вы создали, в любую другую папку.
cp folder_2/file_3.txt folder_2/file_5.json folder_3/
или по отдельности
			cp folder_2/file_3.txt folder_3/
			cp folder_2/file_5.json folder_3/
15) Найти файл по имени
find ./ -name 'file_1.txt'
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает.
tail -f file_2.txt
После ввода команды, в другом окне открываем file_2.txt и в конце добавляем строки, сохраняем.
Изменение отобразится в терминале.
Для фильтрации информации в реальном времени можно использовать grep.
tail -f file_2.txt | grep 'параметр'
17) вывести несколько первых строк из текстового файла
head -n4 file_2.txt
18) вывести несколько последних строк из текстового файла
tail -n4 file_2.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает.
less file_2.txt
прокручивание строк вперед и назад клавишами Enter и y
для выхода Ctrl + z
20) вывести дату и время
date
